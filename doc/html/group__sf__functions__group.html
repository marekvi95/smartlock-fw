◆ SF_ChangeNetworkStandard()

status_t SF_ChangeNetworkStandard ( sf_drv_data_t *  drvData,
sf_net_standard_t  standardSel,
uint8_t  txAttSteps,
sf_xtal_type_t  xtal,
sf_tx_repeat_t  txRepeat,
sf_pa_type_t  paType 
)

This function changes Sigfox networking parameters including up-link and down-link frequency and bit rate.

It enables to set European ETSI standard (default setting), USA FCC standard, Japanese/Korean ARIB standard and South American FCC standard. It uses Change To RCZ1, Change To RCZ2, Change To RCZ3 and Change To RCZ4 commands to set an appropriate standard.

Parameters
drvDataDriver run-time data.
standardSelSelection of a standard defining network communication parameters.
txAttStepsAmount of 0.25dB steps that needs to be subtracted from TX power (default value is 0).
xtalType of used XTAL.
txRepeatAmount of transmissions that will happen on one time invoking of SF_SendPayload function.
paTypeType of internal used PA.
Returns
Status result of the function (kStatus_Success on success).
