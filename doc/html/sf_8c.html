Detailed Description

SIGFOX driver based on AML layer supporting boards based on following NXP parts: OL2361, OL2385.

This module is common for all supported models.

#define SF_ACK_TRANS_RECV_TIMEOUT_US   60000000U

Timeout used to wait for a level of ACK pin. It is intended for the Receive Frame SPI command. The SIGFOX firmware has an internal timeout (60 s).

◆ SF_ACK_TRANS_TIMEOUT_US

#define SF_ACK_TRANS_TIMEOUT_US   15000000U

Timeout used to wait for a level of ACK pin. It is intended for SPI commands causing transmission to SIGFOX network (Send Payload, Send Bit, Send Out of Band, Send Test). The Receive Frame command has higher timeout. Note: waiting for an acknowledge frame can take several second. Max. time is needed for Send Test Mode command (test mode number 5, TxSynth).
