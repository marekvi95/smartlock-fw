\subsection{Detailed Description}
Selection of the timer type that is present on used M\+CU (T\+PM or F\+TM peripheral). 

This option configures the A\+ML timer driver for desired timer type. Use macros defined above (T\+M\+R\+\_\+\+A\+M\+L\+\_\+\+T\+PM or T\+M\+R\+\_\+\+A\+M\+L\+\_\+\+F\+TM).
