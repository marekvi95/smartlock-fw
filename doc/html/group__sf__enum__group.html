enum _sf_status_t

Specific error codes for this driver.

Enumerator
kStatus_SF_SpiAckError 

Received an error from the OL23xx device via SPI. Use the SF_GetErrorCode function to obtain the error code from the last ACK frame received.

kStatus_SF_SpiAckLength 

Size of the receive buffer is smaller than data to be received (indicated by the first byte of a received acknowledgement).

kStatus_SF_SpiTimeout 

SPI communication timeout. Note that some commands take extra time to send data to SIGFOX network (for example Send Payload command).

kStatus_SF_SpiTestFail 

A SPI communication check performed in the SF_Init function failed.
