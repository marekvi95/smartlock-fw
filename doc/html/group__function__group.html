◆ TMR_AML_FillSdkConfig()

status_t TMR_AML_FillSdkConfig ( const tmr_aml_config_t *  amlConfig,
tmr_sdk_config_t *  sdkConfig,
tmr_sdk_pwm_ch_config_t **  chnlPwmParam,
tmr_sdk_outcmp_ch_t **  chnlCmpParam,
tmr_sdk_incpt_ch_t **  chnlIncptParam 
)

Fills SDK specific configuration structure according to independent AML configuration. This function configures S32 SDK or KSDK configuration structure according SDK_VERSION form independent tmrAmlConfig.

Parameters
tmrAmlConfigAML configuration structure, which does not depend on SDK version. It is compatible with S32 SDK and KSDK. Configuration will be used for configure SDK which is currently used. It depends on SDK_VERSION which is define in AML/common_aml.h file.
trmSdkConfigSDK specific configuration, this structure will configure according tmrAmlConfig.
tmr_sdk_pwm_ch_config_t(*)[]pointer to array of tmr_sdk_pwm_ch_config_t for each PWM channel. It is used for generating independent PWM signal on output. Use NULL pointer if PWM is not needed.
tmr_sdk_outcmp_ch_t(*)[]pointer to array of tmr_sdk_outcmp_ch_t for each output compare register. It is used for output compare mode. Use NULL pointer if output compare mode is not needed.
chnlIncptParam(*)[]pointer to array of chnlIncptParam for each input capture register. It is used for input capture mode. Use NULL pointer if output compare mode is not needed.
Returns
Error code (staERR_OK when no error occurred).

< Error code.
